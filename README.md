# Customer Churn Prediction Pipeline

Built an end-to-end machine learning pipeline to predict customer churn using demographic, service, and billing information.

## Project Overview

This project applies data preprocessing, feature engineering, machine learning, and model evaluation techniques to identify customers likely to discontinue services.

## Methods

- Missing value handling
- Feature scaling
- One-Hot Encoding
- Random Forest Classification
- Scikit-learn Pipeline
- Cross Validation

## Results

- Achieved approximately 79% classification accuracy.
- Evaluated performance using precision, recall, F1-score, and cross-validation.
- Automated preprocessing and model training using a reusable machine learning pipeline.
